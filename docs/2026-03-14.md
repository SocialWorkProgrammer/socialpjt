# 2026-03-14 진행 기록

## 핵심 결론
- 프론트엔드는 Django 템플릿 기반 렌더링 API에서 벗어나기 위해
  `/news`, `/community`에 **루트 경로 기준 JSON 전용 API**를 추가하기로 확정.
- 라우팅은 기존 템플릿 URL을 유지하고, SPA 전용 API는 하위 경로로 분리:
  - `GET /news/api/`
  - `GET /news/api/<news_id>/`
  - `GET /community/api/`
  - `GET /community/api/<post_id>/`
- API는 로그인 실패 시 `302` 리다이렉트 대신 **JSON `401`** 반환하고,
  프론트에서 공통 에러 분기로 처리.
- 인증 방식은 기존 합의대로 Django 세션 쿠키 기반 fetch 사용.
